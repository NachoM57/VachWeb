Tipos de Sitios web

Landingpage

Página web diseñada para convertir visitantes en clientes potenciales.

Onepage

Lo más destacado de tu empresa en una única página.

Institucional

Portal web que presenta información sobre una organización o empresa.

Restaurante

Sitio web que muestra menú, ubicación y detalles del restaurante.

Profesionales

Sitio web que destaca habilidades y servicios de profesionales especializados.

Oficios

Sitio web que ofrece servicios especializados y habilidades en diversos oficios.
